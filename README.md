# 🧵 TextileGPT — AI Textile Chatbot (Free with Google Gemini)

An AI-powered chatbot for Fabrics, Yarn, Weaving, Dyeing & Finishing, and Textile Trade — built with Next.js and Google Gemini AI (100% free).

## 🆓 Get Your FREE Gemini API Key (No Credit Card!)

1. Go to https://aistudio.google.com
2. Sign in with your Google account
3. Click "Get API Key" → "Create API key"
4. Copy the key (starts with AIza...)

Free tier: 1,500 requests/day, no credit card needed.

## 🚀 Deploy to Vercel (Free)

1. Upload this folder to a GitHub repository
2. Go to vercel.com → Import your repo
3. Add Environment Variable:
   - Name: GEMINI_API_KEY
   - Value: your key from above
4. Click Deploy!

## 💻 Run Locally

  npm install
  cp .env.example .env.local
  # Add your GEMINI_API_KEY to .env.local
  npm run dev
  # Open http://localhost:3000

## ⚠️ Never commit your API key to GitHub!
